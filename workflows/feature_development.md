# Workflow: Feature Development

1. Founder aligns with Product Manager on direction.
2. Product Manager writes or updates PRD.
3. Founder approves scope.
4. Project Manager decomposes work into tasks.
5. Tasks are assigned to specialist agents.
6. Engineer implements on branch.
7. QA reviews against standards.
8. CI runs lint, typecheck, tests, and build.
9. Founder or designated approver approves merge.
10. Release notes and docs are updated as needed.
